# weed-cowork

Document collaboration tools for manufacturing environments.

## Skills

### issue-report
Manufacturing floor issue documentation — converts unstructured text (conversations, memos, error logs) into structured troubleshooting documents stored in Notion.

- `/issue-report` — Create issue document from current conversation
- `/issue-report <text>` — Create issue document from text
- `/issue-report search <keyword>` — Search existing issues
- `/issue-report list` — List all issues

All documents are stored exclusively in Notion databases.

## Installation

This plugin is available through the `weed-plugins` marketplace.

## Requirements

- Notion MCP server configured in `~/.claude/settings.json`
